## Bug 1: XP Sync — ProfileForm tampilkan XP stale dari DB

### Root Cause

`ProfileForm.tsx:90` tampilkan `profile.xp` yang dibaca dari `profiles` table di Supabase.
`profiles.xp` tidak pernah di-update saat achievement di-award — hanya `achievements` table yang berubah.
`useAchievements().totalXp` hitung XP dari `achievements` table secara real-time (benar).
Hasilnya: Profile page tampilkan XP lama, Dashboard tampilkan XP benar.

### Fix

Ganti `profile.xp` di `ProfileForm` dengan `totalXp` dari `useAchievements()`.
Tidak perlu update DB — cukup baca dari sumber yang benar di UI.

## Bug 3: BookmarksTab — cameraPosition dipakai sebagai cameraTarget

### Root Cause

`BookmarksTab.tsx:41–48` — fallback branch:

```typescript
} else if (b.cameraPosition) {
  setCameraTarget(
    new THREE.Vector3(b.cameraPosition.x, b.cameraPosition.y, b.cameraPosition.z)
  );
}
```

`cameraPosition` = posisi kamera (dari mana kamera melihat).
`cameraTarget` = titik yang dilihat kamera (fokus/objek).

Menggunakan `cameraPosition` sebagai `cameraTarget` artinya kamera akan "melihat" ke posisi lama kamera itu sendiri — bukan ke objek yang seharusnya.

### Fix

Jika bookmark tidak punya `cameraTarget`, jangan set camera target sama sekali — biarkan target default atau set null. Camera akan tetap di posisi yang benar setelah `setScale` dipanggil.

### Task 3.1 — Fix BookmarksTab.tsx

**File:** `src/components/profile/BookmarksTab.tsx`

- [ ] **Step 1: Baca file**
      Konfirmasi baris 37–49 di `handleRestore`.

- [ ] **Step 2: Hapus fallback cameraPosition branch**

  Ganti:

  ```typescript
  if (b.cameraTarget) {
    setCameraTarget(
      new THREE.Vector3(b.cameraTarget.x, b.cameraTarget.y, b.cameraTarget.z),
    );
  } else if (b.cameraPosition) {
    setCameraTarget(
      new THREE.Vector3(
        b.cameraPosition.x,
        b.cameraPosition.y,
        b.cameraPosition.z,
      ),
    );
  }
  ```

  Menjadi:

  ```typescript
  if (b.cameraTarget) {
    setCameraTarget(
      new THREE.Vector3(b.cameraTarget.x, b.cameraTarget.y, b.cameraTarget.z),
    );
  } else {
    setCameraTarget(null); // reset ke default, jangan pakai cameraPosition
  }
  ```

- [ ] **Step 3: Hapus import THREE jika tidak lagi dipakai**
      Cek apakah `THREE` masih dipakai di file — jika tidak, hapus `import * as THREE from "three"`.
      `THREE` masih dipakai di baris yang tersisa (`new THREE.Vector3`), jadi import tetap perlu.

- [ ] **Step 4: Verifikasi TypeScript**
  ```bash
  npx tsc --noEmit
  ```
